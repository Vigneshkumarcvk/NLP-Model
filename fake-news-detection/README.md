# Fake News Detection using NLP

This project builds an NLP model to detect Fake News using:

- TF-IDF + Logistic Regression
- LSTM Deep Learning
- BERT Transformer Model

Dataset: Fake News Dataset (4000 rows)

Development Environment:
- Google Colab
- Python
- Scikit-learn
- TensorFlow
- HuggingFace Transformers
